## 物理笔记

### 量子力学基础

薛定谔方程：

$$
i\hbar \frac{\partial}{\partial t} \Psi(\mathbf{r}, t) = \hat{H} \Psi(\mathbf{r}, t)
$$

其中，\(\hat{H}\) 是哈密顿算符，\(\Psi\) 是波函数。

### 热力学第二定律

熵增原理：

$$
\Delta S \geq 0
$$

对于绝热过程，系统的熵永不减少。
